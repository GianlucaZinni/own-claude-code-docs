# Sources

La carpeta `sources/` no forma parte del sitio publicado. Se usa para trazabilidad, curacion y gobierno editorial.

## Reglas

- Registrar cada fuente externa en `registry.yaml` antes de derivar contenido.
- Guardar solo metadatos, notas y assets explicitamente permitidos.
- No subir material crudo pesado ni contenido con dudas de licencia.
- Mantener los paths locales relativos a la raiz del repo.

## Carpetas ignoradas

- `sources/raw/`: materiales crudos temporales o pesados.
- `sources/tmp/`: trabajo transitorio de curacion.

Ambas rutas estan ignoradas por Git.
