## Que es / para que sirve

Esta pagina resume la notacion minima para operar con Claude Code: slash commands, referencias con `@`, flags de CLI y recursos expuestos via MCP.

## Cuando usarlo

- Cuando estas aprendiendo a pedir acciones de forma precisa.
- Cuando quieres documentar una sintaxis comun para el equipo.

## Riesgos o limites

- La misma notacion puede significar cosas distintas segun el contexto si no se documenta bien.
- Referenciar recursos MCP sin aclarar scope y origen complica la trazabilidad.
