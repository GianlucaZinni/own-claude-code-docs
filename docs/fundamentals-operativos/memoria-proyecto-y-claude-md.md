## Que es / para que sirve

La memoria util en Claude Code combina contexto de sesion, archivos del proyecto y contratos explicitos como `CLAUDE.md`.

## Cuando usarlo

- Cuando quieres que un repo exprese convenciones estables.
- Cuando necesitas separar conocimiento durable de decisiones de sesion.

## Riesgos o limites

- Un `CLAUDE.md` vago o excesivo agrega ruido en vez de claridad.
- La memoria automatica sin criterio puede fijar informacion obsoleta.
