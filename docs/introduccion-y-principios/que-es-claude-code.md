## Que es / para que sirve

Claude Code es un entorno agentico para trabajar sobre codigo, archivos y terminal con un loop continuo de contexto, accion y verificacion.

## Cuando usarlo

- Cuando necesitas asistencia activa sobre un repositorio real.
- Cuando quieres pasar de preguntas sueltas a ejecucion guiada por objetivos.

## Riesgos o limites

- No reemplaza el criterio tecnico ni el ownership humano.
- Sin permisos, contexto y verificacion bien definidos puede producir cambios de baja calidad.
