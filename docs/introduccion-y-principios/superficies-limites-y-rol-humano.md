## Que es / para que sirve

Claude Code existe en varias superficies operativas, pero siempre requiere que alguien defina objetivos, limites, aprobaciones y criterios de aceptacion.

## Cuando usarlo

- Cuando necesitas distinguir CLI, IDE, web, CI/CD y control remoto.
- Cuando quieres dejar claro que el usuario sigue siendo responsable del resultado.

## Riesgos o limites

- Cambiar de superficie no cambia la necesidad de gobierno tecnico.
- Sin rol humano claro, se confunden automatizacion con delegacion completa.
