## Que es / para que sirve

La ergonomia del terminal define cuanto contexto visible, control y velocidad tienes durante una sesion con Claude Code.

## Cuando usarlo

- Cuando preparas tu entorno diario.
- Cuando quieres estandarizar keybindings, status line y multiline input.

## Riesgos o limites

- Una terminal mal configurada dificulta diagnosticar costos, contexto y estado del repo.
- No todos los atajos o modos tienen el mismo soporte por plataforma.
