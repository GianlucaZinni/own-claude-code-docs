## Que es / para que sirve

Esta pagina ordena el primer arranque de Claude Code: instalacion, autenticacion y comprobaciones basicas antes de usarlo en serio.

## Cuando usarlo

- Cuando arrancas desde cero.
- Cuando quieres un baseline comun para documentar setup por equipo.

## Riesgos o limites

- Un setup incompleto suele aparecer mas tarde como fallo de permisos, terminal o integracion.
- La forma de autenticacion impacta que superficie y features estan disponibles.
