## Que es / para que sirve

Las skills permiten empaquetar instrucciones y workflows reutilizables para que Claude Code los invoque de forma consistente.

## Cuando usarlo

- Cuando una secuencia de trabajo aparece una y otra vez.
- Cuando quieres convertir prompts ad hoc en artefactos compartibles.

## Riesgos o limites

- Una skill demasiado amplia se vuelve ambigua.
- Una skill sin criterio de activacion claro no mejora realmente la operacion.

## Fuentes utilizadas

- `anthropic-claude-code-docs-site`
- `anthropic-skills`

## Defaults recomendados

- Mantener una responsabilidad por skill.
- Versionar skills de equipo igual que cualquier otra pieza operativa.
