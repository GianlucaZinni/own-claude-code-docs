## Que es / para que sirve

`CLAUDE.md` funciona como contrato operativo del repositorio: explicita convenciones, expectativas y limites que Claude Code debe respetar.

## Cuando usarlo

- Cuando un repo necesita reglas claras y repetibles.
- Cuando quieres convertir conocimiento tacito en instrucciones estables.

## Riesgos o limites

- Si el archivo intenta documentarlo todo, deja de ser util.
- Si no se mantiene, pasa a competir contra la realidad del repo.

## Fuentes utilizadas

- `anthropic-claude-code-docs-site`

## Defaults recomendados

- Mantenerlo corto, operativo y revisable.
- Usarlo para decisiones estables, no para notas temporales de una sola sesion.
