## Que es / para que sirve

Esta pagina cubre el uso de Claude Code en automatizacion y revision sobre GitHub: ejecucion en CI, menciones, PRs y code review asistido.

## Cuando usarlo

- Cuando tu equipo trabaja sobre GitHub y quiere integrar Claude Code al flujo normal.
- Cuando necesitas entender la frontera entre revision automatizada y decision humana.

## Riesgos o limites

- Automatizar revision sin criterios claros genera ruido en vez de senales utiles.
- Integrar CI sin controlar permisos y costos puede abrir riesgos operativos.
