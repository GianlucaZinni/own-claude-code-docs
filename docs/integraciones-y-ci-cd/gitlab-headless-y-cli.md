## Que es / para que sirve

Esta pagina agrupa las superficies de integracion no centradas en GitHub: GitLab, modos headless y uso CLI en automatizaciones propias.

## Cuando usarlo

- Cuando GitHub no es la unica superficie del equipo.
- Cuando necesitas pensar Claude Code como componente de una automatizacion mas amplia.

## Riesgos o limites

- Ejecutar en modo headless sin observabilidad vuelve opaco el diagnostico.
- Cambiar de superficie no elimina la necesidad de permisos, limites y verificacion.
